<i>Permission Manager X</i> — это небольшое приложение для управления разрешениями и AppOps приложений, установленных на вашем устройстве. Оно развивалось от shell скрипта до графического интерфейса для моих личных нужд. После обновления прошивки или смены устройства, обзор всех установленных приложений на предмет предоставленных разрешений и отзыва ненужных занимает длительное время (в конце концов, <b><i>приватность имеет значение</i></b>). Для решения этой задачи вы можете настроить желаемые <b>целевые значения</b> разрешений, которые можно быстро забэкапить и восстановить. Цветные полоски слева показывают состояние разрешений относительно целевых, что позволяет быстро визуально оценить настройки приложений.

<b>Разрешения манифеста</b> обычно называются просто «разрешениями» (например, «Хранилище», «Камера»). <b>AppOps</b> (операции приложения) это инфраструктурные разрешения, которые Android использует как ядро управления доступом. С каждым выпуском Android разрешения манифеста становятся более зависимыми от AppOps. Поэтому интересно контролировать оба типа разрешений одновременно и смотреть, как они относятся друг к другу.

Кратко говоря, <b>AppOps</b> обеспечивают тонкий контроль над многими разрешениями манифеста. Кроме того, они предоставляют дополнительный контроль, такой как: работа в фоне, вибро, доступ к буферу обмена и т. д. Изучите приложение, чтобы узнать больше.

<big><b>Особенности:</b></big>

Используя <b>Permission Manager X</b>, для каждого установленного приложения, на одном экране, вы можете:

* Просматривать, выдавать или отзывать разрешения манифеста
* Просматривать разрешения AppOps и выбирать один из множества режимов
* Устанавливать желаемое целевое значение для каждого изменяемого разрешения

<big><b>Требуемые привилегии/разрешения:</b></big>

* Чтобы <i>Permission Manager X</i> работал у вас наилучшим образом, на устройстве должен быть <b>root</b>, либо вам необходимо включить <b>ADB по сети</b>.
* <code>android.permission.INTERNET</code> требуется для использования ADB через сеть. Никаких внешних соединений не производится.
